```
char * to_upper(char* s) {
	int i = 0;
	while(*(s + i)) {
		if ((*(s + i) >= 'a') && (*(s + i) <= 'z'))
			*(s + i) -= ('a'-'A');
		i++;
	}
	return s;
}

char* to_upper1(char* s) {
	char* save = s;
	while (*s) {
		if ((*s >= 'a') && (*s <= 'z'))
			*(s) -= ('a' - 'A');
		s++;
	}
	return save;
}
```
아래 코드가 디스어셈블리 시 메모리 접근 연산이 적어지므로 좀 더 효율적
